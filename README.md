# Spotify Recently Played ETL Pipeline

## Overview
This project extracts "Recently Played" tracks from the Spotify API, transforms the data using Pandas, and loads it into a SQL Data Warehouse.

## Setup Instructions
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`.
3. Install dependencies: `pip install -r requirements.txt`.
4. Create a `.env` file with your Spotify `CLIENT_ID`, `CLIENT_SECRET`, and `TOKEN`.
